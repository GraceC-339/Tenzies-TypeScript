## Features

- Interactive dice rolling
- Click to hold dice
- Win celebration with confetti
- Responsive design
- TypeScript for type safety

## How to Play

1. Roll the dice
2. Click dice to "hold" them at their current value
3. Keep rolling until all dice show the same number
4. Celebrate your win!

## Technologies Used

- React
- TypeScript
- Vite
- CSS

## Getting Started

```bash
npm install
npm run dev
```
